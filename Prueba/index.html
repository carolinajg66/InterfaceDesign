Web Page

This page will talk about the css, but first we will talk about what it is, and what it is used for. The css is used to make a web page, for those who do not know what a web page is, a web page is a document or electronic information capable of containing text, sound, video, programs, links, images and many other things and can provide access to other web pages through hypertext links, and is composed of html and css, which are two of the languages used to create them, although other languages are also used, these two are essential to create one. To know more about html and css, you can look at the links where it is explained in more detail what is each thing. In the css, you will find five tutorials to put this language into practice.
